# UI

Este documento contiene los diseños de interfaz de usuario (UI) para la aplicación de gestión de medicamentos y recordatorios. Incluye las pantallas principales del sistema, desde el inicio de sesión hasta la gestión completa de medicamentos y recordatorios.

*link*: https://www.figma.com/design/87nLGcZ0jJO5aRZxcH9YZw/UI-s-MEDICOD?node-id=0-1&t=EtjAFffJ9n7ZpUU8-1

# Iniciar Sesión

Pantalla de inicio de sesión donde los usuarios pueden acceder a su cuenta utilizando sus credenciales.

![Iniciar Sesión](https://i.imgur.com/czGTsM4.png)

# Crear Cuenta

Pantalla de registro de nuevos usuarios donde se pueden crear cuentas con información personal básica.

![Crear Cuenta](https://i.imgur.com/vveR9r0.png)

# Home

Pantalla principal del sistema donde los usuarios pueden ver un resumen de sus medicamentos, recordatorios próximos y acceder rápidamente a las funciones principales.

![Home](https://imgur.com/xSErGvS.png)

# Recordatorios

Vista principal de recordatorios donde los usuarios pueden ver todos sus recordatorios activos y gestionarlos.

![Recordatorios](https://i.imgur.com/KBniQYn.png)

# Crear un Recordatorio

Formulario para crear nuevos recordatorios con opciones de personalización de horarios y tipos de recordatorio.

![Crear un Recordatorio](https://i.imgur.com/hhw1a9J.png)

# Editar un Recordatorio

Pantalla de edición de recordatorios existentes, permitiendo modificar horarios, tipos y configuraciones.

![Editar un Recordatorio](https://imgur.com/Ax5V9ue.png)

# Crear un Medicamento

Formulario para agregar nuevos medicamentos al sistema con información detallada como dosis, frecuencia y horarios.

![Crear un Medicamento](https://i.imgur.com/fbTCLYE.png)

# Editar un Medicamento

Pantalla de edición de medicamentos existentes, permitiendo actualizar información, dosis y configuraciones.

![Editar un Medicamento](https://i.imgur.com/xxT1S75.png)
